Entire tax calculation engine as a CLI

186 registered nodes. Every IRS form piped as JSON. Inspect schemas, create returns, compute taxes — all from your terminal.

opentax --help
$ opentax --help Filed Open Tax Engine CLI Usage: opentax <command> <subcommand> [options] Commands: • opentax node list List all registered nodes • opentax node inspect --node_type <type> Inspect a node's input schema and output nodes • opentax return create --year <year> Create a new tax return • opentax return get --returnId <id> Get a return's computed line items • opentax return export --returnId <id> --type mef|pdf [--force] [--output <path>] Export a return as MEF XML or filled IRS PDF • opentax form add --returnId <id> --node_type <type> '{...}' Add a form entry to a return • opentax form list --returnId <id> [--node_type <type>] List all form entries in a return • opentax form get --returnId <id> --entryId <id> Get a specific form entry by ID • opentax form update --returnId <id> --entryId <id> '{...}' Update a form entry's data • opentax form delete --returnId <id> --entryId <id> Delete a form entry • opentax return validate --returnId <id> [--format text|json] Validate a return against MeF business rules • opentax node graph --node_type start [--depth <n>] [--json] View node dependency graph (Mermaid or JSON) Global flags: --help, -h Show help --json Output as JSON
